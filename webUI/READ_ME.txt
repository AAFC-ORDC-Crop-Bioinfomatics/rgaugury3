==== How to install ====

1. install python2.7  "sudo apt-get install python2.7"
2. install python-pip "sudo apt-get install python-pip"
5. install python-dev "sudo apt-get install python-dev" to install psutil
3. install modules    "sudo pip install -r requirements.txt"
4. install database   "sudo apt-get install sqlite3"
5. initialize database    "./initializeDB.py"

========================
css files are generated by "stylus". If you want to modify css,
you should install stylus first, and compile the corresponding stylus file to
generate the css file.

How to install "stylus"

1. sudo apt-get install npm
2. sudo npm install stylus -g
3. sudo ln -s /usr/bin/nodejs /usr/bin/node
